Ferramenta de Correção de Status ADMIN

Instruções

Esta ferramenta permite corrigir o status de um usuário para ADMIN ou outro status quando a atualização normal não estiver funcionando.

Use esta ferramenta apenas quando:

  • O status do usuário estiver sendo mostrado incorretamente
  • As permissões de administrador não estiverem funcionando corretamente

Importante: Você precisa estar logado como administrador para usar esta ferramenta.

Alternativa: Usar SQL Direto

Se a abordagem via API não funcionar, você pode executar o seguinte SQL diretamente no SQL Editor do Supabase:

-- Substitua os valores
DO $$
DECLARE
  target_user_id UUID := '0b671175-c282-4e3b-b53d-e82468d315e5';
  new_status TEXT := 'ADMIN';
BEGIN
  -- Desativar RLS temporariamente
  SET session_replication_role = 'replica';
  
  -- Atualizar status
  UPDATE public.users 
  SET status = new_status, updated_at = NOW() 
  WHERE id = target_user_id;
  
  -- Reativar RLS
  SET session_replication_role = 'origin';
END
$$;
